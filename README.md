## ⚙️ 전략 1 — 무한매매 (INFINITE)

승승장군 무한매수법 구글 시트 v4.0 완전 동기화

### 큰수(大數) 매수 원리

```
큰수 = 현재가 × (1 + 큰수%)  ← 동시호가 주문 상한가
수량 = 예산 ÷ 큰수            ← 보수적 수량 계산
실제 체결 = 종가 (큰수보다 낮게)
```

### 매수 전략 (15:10~15:20 동시호가)

| 상황 | 전략 |
|------|------|
| 새출발 (qty=0) | 큰수 가격으로 1회차 + 줍줍 5개 |
| 전반전 (T < n/2) | min(별값,큰수) 50% + min(평단,큰수) 50% |
| 후반전 (T ≥ n/2) | min(별값,큰수) 100% |
| 줍줍 | 예산÷(n+1~5), 큰수 미만만, 최대 5개 |

### 매도 전략 (장중 1분 감시)

| 조건 | 액션 |
|------|------|
| 현재가 ≥ 별값 | 1/4 수량 지정가 매도 |
| 현재가 ≥ 목표가 | 전량 시장가 매도 |

## ⚖️ 전략 2 — V-REV (Value Rebalancing)

```
목표 밸류 = 설정값 (예: 3,000,000원)
현재 밸류 = 보유수량 × 현재가

편차 < -밴드%  →  Buy1(현재가 60%) + Buy2(현재가-0.5% 40%)
편차 > +밴드%  →  Pop1(평단×1.006) + Pop2(평단×1.005)
```

매일 09:10 1회 실행. 목표 밸류 근처로 자동 수렴.

## ⚡️ 전략 3 — AVWAP 퀀트 엔진 (신규 v5.0)

승승장군 V44~V79 옴니 매트릭스 퀀트 엔진 국내 정규장 버전

### 국내 운영 시간

| 시각 | 상태 |
|------|------|
| 09:00~09:30 | 🛡️ 타임쉴드 (개장 휩소 차단) |
| 09:30~14:30 | ⚔️ 교전 가능 (진입·청산) |
| 14:30~15:20 | 🚫 신규 진입 금지 |
| 15:20 | 🔔 타임스탑 강제 전량 청산 |

### 진입 조건 (듀얼 모멘텀)

```
① 당일 VWAP > 전일 VWAP    ← 모멘텀 상승
② 현재 VWAP > 5분 평균 VWAP ← 단기 추세
③ ATR5 잔여체력 ≥ 2.0%      ← 오버슈팅 차단
→ 3조건 만족 시 진입
```

### 다이나믹 목표가 (ATR5 기반)

| 잔여체력 | 목표 수익률 |
|---------|-----------|
| ≥ 5% | 5.0% |
| ≥ 4% | 4.0% |
| ≥ 3% | 3.0% |
| 기타 | **2.0%** (RR 최저 방어막) |

### 청산 규칙

```
익절 도달  →  청산 → 쿨다운 5분 → 재진입 (다중 출장)
하드스탑 -8%  →  즉시 손절 + 당일 완전 동결
타임스탑 15:20  →  전량 강제 청산
```

### 백테스트 (KODEX 반도체TOP10레버리지 시뮬레이션, 2년)

| | 단리 | 복리 1백만 | 복리 3백만 |
|-|------|-----------|-----------|
| 수익률 | — | **+301.6%** | **+374.9%** |
| 승률 | 64.7% | 64.7% | 64.7% |
| 총 출장 | 374회 | 374회 | 374회 |
| 평균 익절 | +1.90% | +1.90% | +1.90% |
| 하드스탑 | 0회 | 0회 | 0회 |

> ⚠️ 시뮬레이션 데이터 기반. 실제 결과와 다를 수 있습니다.

### 조회
| 명령어 | 기능 |
|--------|------|
| `/sync` | 통합 지시서 (T값·큰수·별값·주문계획·매도계획) |

### 제어
| 명령어 | 기능 |
|--------|------|

## 🛠️ 설치 및 설정

### 1. config.json (data/ 폴더)

```json
{
  "SYMBOLS": [
    {
      "code": "122630",
      "name": "KODEX 레버리지",
      "mode": "INFINITE",
      "active": true,
      "allocation_krw": 3000000,
      "split_count": 10,
      "target_profit_pct": 5.0,
      "large_num_pct": 15.0,
      "vrev_band_pct": 3.0,
      "daily_buy_limit_krw": 300000,
      "avwap_budget": 0
    }
  ],
  "TRADE_MODE": "MOCK",
  "BOT_PAUSED": false
}
```

**avwap_budget > 0** 이면 AVWAP 자동 활성화.
텔레봇 `/settlement` → `⚡️ AVWAP OFF` 버튼으로도 토글 가능.

### 2. 배포

```bash
deploy_kiw_moohan
```

### 3. 백테스트 실행

```bash
cd ~/kiw_moohan_trader
source etf_venv/bin/activate
python3 avwap_backtest.py
```
